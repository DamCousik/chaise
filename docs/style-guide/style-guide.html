Deriva UI Style Guide

This style guide communicates the meaning of every component in the Chaise application.

Typography

Typography is the concept of having highly legible headers and body paragraphs, and clear labels.

The Chaise application uses Helvetica Neue as the primary font for all text. The following is the default CSS styles for all text: 

body {
  font-family: "HelveticaNeue-Light", "Helvetica Neue Light", "Helvetica Neue", Helvetica, Arial, "Lucida Grande", sans-serif;
  font-size: 14px;
  line-height: 20px;
}
