Chaise Style Guide

This style guide communicates the meaning of every component in the Chaise application.

Typography

Typography is the concept of having highly legible headers and body paragraphs, and clear labels.

Font

The Chaise application uses Helvetica Neue as the primary font for all text.

Body

Chaise's global default font-size is 14px, with a line-height of 1.428. This is applied to the <body> and all paragraphs
